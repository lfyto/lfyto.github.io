# 我的个人网站

这是一个包含HTML作业集成的个人网站项目。

## 项目结构

- `index.html` - 网站首页
- `html_homework_manager.html` - HTML作业管理页面
- `homeworks/` - 所有的HTML作业文件


## 功能特点

1. 动态的HTML作业导航系统
2. 支持单文件和多文件夹作业
3. 星点背景和鼠标轨迹特效
4. 响应式设计，支持移动设备
5. 作业自动分类和管理

## 本地运行

直接将`index.html`或`html_homework_manager.html`在浏览器中打开即可。
